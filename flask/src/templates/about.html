Journal Reader

{% endblock header %} {% block content %}

Parses a properly formatted .jrl file and enables reading it in a more
comfortable manner with colors that are not black and white like a terminal
window. Be sure to put on some music that matches the feels.

There is an encryption algorithm implemented, guarded by a key. If you are
looking for one, chances are that you're somewhere you shouldn't be. Be a
good person and respect other's privacy.

The code is open source! Figure out the algorithm and brute force the
key. Or better if a ton of data is available, make out the key from the
data like a real cryptographer. The algorithm is quite simple.

This project is open source.
